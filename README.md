# EV-D68 Nextstrain Community Build

## About repository

This repository contains the data for constructing the EV-D68 Nextstrain phylogeny used within 
[Household Transmission of Enterovirus D68 in the United States, 2022-2024]() *Link to be added at a later date*. 

## Repository contents
auspice/ - Contains json files used to generate phylogenies<br>
README.md - Markdown file providing context of repository contents

## How to run

Please upload json file to [auspice](https://auspice.us/) or access via [Nextstrain Community Builds](https://nextstrain.org/community/uwvirology-ngs/EV-D68/HouseholdTransmission/).
